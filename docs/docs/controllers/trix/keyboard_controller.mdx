## Purpose

A controller that adds the ability for Trix editors to format text as a keyboard button or shortcut.

Fire and forget. The only configuration this controller, is a rule in your CSS for how it should look.
Mount it on your `<trix-editor>` element, add a style rule for

```css
.trix-button.trix-button--icon.trix-button--icon-kbd::before {
  background-image: url("data:image/svg+xml;base64,...");
}

kbd {
  font-family: monospace;
  font-weight: bold;
  border-radius: 4px;
  background: grey;
  box-shadow: 0 0 0 1px rgba(0,0,0,.1);
}
```
and you're good to go.

## Side Effects

The controller will inject a button into the Trix toolbar, and modify the Trix instance to be able to handle formatting text
with `<kbd>` blocks.
